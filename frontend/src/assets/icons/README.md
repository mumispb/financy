# Icons from Figma Style Guide

This folder should contain SVG icons exported from the Figma style guide.

## Required Icons

Please export the following icons from the Figma style guide (node-id: 3-377) and place them in this folder:

- `mail.svg` - Email/Mail icon
- `lock.svg` - Lock/Password icon  
- `eye.svg` - Eye icon (show password)
- `eye-off.svg` - Eye off icon (hide password)
- `user.svg` - User icon
- `user-plus.svg` - User plus icon (signup)
- `log-in.svg` - Login icon

## Export Instructions

1. Open the Figma style guide: https://www.figma.com/design/qHdzyL5MiZvEWidmthcfds/Financy--Community-?node-id=3-377
2. Select each icon component
3. Right-click → Copy/Paste as → Copy as SVG
4. Save as SVG file in this folder with the names listed above

Alternatively, you can use Figma's export feature:
1. Select the icon
2. In the right panel, click Export
3. Choose SVG format
4. Export and save to this folder
